홈(대시보드) 기초 역사 서비스 자격증 퀴즈

☁️ 클라우드 컴퓨팅의 이해

클라우드는 컴퓨터 자원을 소유하는 대신, 인터넷을 통해 필요한 만큼 빌려 쓰는 기술적 패러다임입니다.

1. 클라우드의 핵심 6가지 장점

  • 자본 비용을 가변 비용으로 대체: 서버를 미리 살 필요가 없습니다.
  • 규모의 경제: 수많은 사용자가 이용하므로 단가가 저렴해집니다.
  • 추측성 용량 산정 불필요: 트래픽이 늘면 즉시 서버를 늘릴 수 있습니다.
  • 속도 및 민첩성 개선: 몇 분 만에 서버 배포가 가능합니다.
  • 비용 절감: 데이터 센터 운영 비용을 걱정할 필요가 없습니다.
  • 몇 분 만에 전 세계 배포: 클릭 몇 번으로 전 세계에 서비스를 출시합니다.

2. 클라우드 서비스 모델 비교

IaaS PaaS SaaS
인프라 수준 제공 (서버, 네트워크) 개발 환경까지 제공 (OS 포함) 완성된 소프트웨어 제공
예: AWS EC2, S3 예: AWS Elastic Beanstalk 예: Gmail, Dropbox

3. 배포 모델

  • 퍼블릭 클라우드: 누구나 가입해서 이용 가능 (AWS, Azure)
  • 프라이빗 클라우드: 특정 기업 내에서만 운영
  • 하이브리드: 퍼블릭과 프라이빗을 혼합하여 사용
